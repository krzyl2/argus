## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Information disclosure | Insecure h2c channel on loopback (local mode) | accept | Plaintext is confined to 127.0.0.1 within a single container (same trust domain as the orchestrator). No LAN exposure. mTLS is retained for any non-loopback https endpoint. |
| T-02-02 | Spoofing / Tampering | Scheme discrimination in IsLocalMode | mitigate | Insecure path is gated on http scheme OR loopback host only; any https endpoint falls through to the unchanged mTLS path with CustomRootTrust CA pinning. A remote https endpoint can never silently downgrade to insecure. |
| T-02-03 | Denial of service | h2c switch (process-global AppContext) | accept | Switch only permits cleartext HTTP/2; it does not force it. Remote https channels still negotiate TLS. Low risk for a single-operator self-hosted deployment. |

No package-manager installs in this plan → no supply-chain (T-*-SC) checkpoint required.
