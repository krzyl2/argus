Combine model (AUTHORITATIVE — from CONTEXT.md, do not simplify):
  resolved = (entities matching include-globs − entities matching exclude-globs)
             ∪ manually-checked − manually-unchecked
  - When there are NO include patterns: ALL snapshot entities are the base candidate set.
  - Excludes remove from the pattern-selected set.
  - manually-checked are ADDED after excludes (manual check overrides an exclusion).
  - manually-unchecked are REMOVED last (manual uncheck overrides an inclusion). Manual wins.

Signature to create (orchestrator/Argus.Orchestrator/Config/GlobExpander.cs, public static):
```csharp
public static HashSet<string> Resolve(
    IReadOnlyList<HaSensorEntry> snapshot,
    IEnumerable<string> includePatterns,
    IEnumerable<string> excludePatterns,
    IEnumerable<string> manuallyChecked,
    IEnumerable<string> manuallyUnchecked);
```

Glob primitive (BCL, net8.0, ZERO new deps): `using System.IO.Enumeration;`
`FileSystemName.MatchesSimpleExpression(pattern, input, ignoreCase: true)` — supports `*` and `?`.
Analog for pure-static style: NetDaemonHaEventSource.SelectDiscoverableSensors (invariant, HashSet OrdinalIgnoreCase).
