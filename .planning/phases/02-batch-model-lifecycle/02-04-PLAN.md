## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-04-01 | Denial of Service | PeriodicTimer burst on slow tick | mitigate | PeriodicTimer.WaitForNextTickAsync skips accumulated ticks (unlike Task.Delay) — no burst after downtime |
| T-02-04-02 | Denial of Service | Infinite InfluxDB query hang | mitigate | InfluxDB.Client QueryAsync propagates CancellationToken from stoppingToken; worker stops cleanly |
| T-02-04-03 | Tampering | ScoreBatch verdict IsAnomaly field | accept | Batch path publishes raw is_anomaly from detector; hysteresis is streaming-path only; batch path users accept this |
| T-02-04-04 | Denial of Service | Per-entity gRPC exception crash worker | mitigate | Per-entity try/catch logs BatchSchedulerError and continues; worker never dies from single entity failure |
| T-02-04-SC | Tampering | InfluxDB.Client / Grpc.Net.Client installs | accept | Both already in csproj from Phase 1; no new installs |
</threat_model>
