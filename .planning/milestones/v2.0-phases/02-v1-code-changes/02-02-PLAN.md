<objective>
Implement CODE-02 and CODE-03: make the detector bind address configurable via `ARGUS_GRPC_BIND` and the model root configurable via `ARGUS_MODEL_ROOT`. Both default to their v1 values when unset (`[::]` and `/var/argus/models`).

Purpose: The add-on runs the detector in local mode bound to `127.0.0.1` and persists models under `/data/models`. Today bind address (`[::]`) is hardcoded in `server.py` and the model root (`/var/argus/models`) is hardcoded in `model_store.py`. This is the detector half of Phase 2 (Python codebase; independent of the orchestrator half in plan 01 — zero file overlap, same wave).

Output: Two new config fields, `server.py` consuming both, and unit tests proving the bind override and model_root pass-through with backward-compatible defaults.
</objective>
