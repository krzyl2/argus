mTLS channel (RESEARCH Pattern 2, D-18) — implement EXACTLY:
```csharp
var caCert = X509CertificateLoader.LoadCertificateFromFile("deploy/certs/ca.crt"); // or new X509Certificate2(...)
var clientCert = X509Certificate2.CreateFromPemFile("deploy/certs/client.crt", "deploy/certs/client.key");
var handler = new HttpClientHandler();
handler.ClientCertificates.Add(clientCert);
handler.ServerCertificateCustomValidationCallback = (_, cert, chain, _) => {
    chain!.ChainPolicy.TrustMode = X509ChainTrustMode.CustomRootTrust;
    chain.ChainPolicy.CustomTrustStore.Add(caCert);
    return chain.Build(cert!);
};
var channel = GrpcChannel.ForAddress("https://<gpu-host>:50051",
    new GrpcChannelOptions { HttpHandler = handler });
// ONE channel per process.
```

Health gate (RESEARCH grpc.health.v1 .NET poll):
```csharp
var healthClient = new Health.HealthClient(channel);
var resp = await healthClient.CheckAsync(new HealthCheckRequest { Service = "argus.v1.DetectorService" },

if (resp.Status != HealthCheckResponse.Types.ServingStatus.Serving) throw ...;
```
