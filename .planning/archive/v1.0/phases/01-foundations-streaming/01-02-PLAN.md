## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| orchestrator (client) → detector gRPC :50051 | Untrusted-until-mTLS network input crossing host boundary |
| Docker host → mounted certs volume | Private key material exposed read-only to container |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Spoofing | gRPC client identity | mitigate | `add_secure_port` with mTLS server credentials requiring client certificate auth (require_client_auth=True); insecure port only when no TLS config (unit tests) |
| T-02-02 | Denial of Service | ScoreStream unbounded request iterator | mitigate | ThreadPoolExecutor max_workers=10 caps concurrency; per-request work is O(1) score; `context.is_active()` check exits dead streams |
| T-02-03 | Information Disclosure | structured logs | mitigate | Log only entity_id, score, latency_ms, detector — no raw secrets, no cert paths' contents |
| T-02-04 | Tampering | container supply chain | accept | Pinned base image python:3.12-slim-bookworm + pinned pip requirements; ASVS L1 pinning control |
</threat_model>
