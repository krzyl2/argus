    Module docstring must note: stateless, no fit(), period=1440 for 60s-interval sensors, minimum 2880 points (48h), and that with 24h window the guard always fires in Phase 2 (expected, not a bug).

    Imports: from __future__ import annotations; import numpy as np; from statsmodels.tsa.seasonal import STL.

    Module-level: _PERIOD_DAILY = 1440.

    Class StlDetector:
    - score_batch(self, values: list[float], period: int = _PERIOD_DAILY) -> tuple[list[float], str | None]:
      - n = len(values)
      - if n < 2 * period: return [], f"insufficient history: got {n} points, need >= {2 * period}"
      - x = np.array(values, dtype=float)
      - result = STL(x, period=period, robust=True).fit()   # robust=True mandatory
      - residuals = np.abs(result.resid)
      - rng = residuals.max() - residuals.min()
      - if rng == 0: return [0.0] * n, None
      - scores = ((residuals - residuals.min()) / rng).tolist()
      - return scores, None
